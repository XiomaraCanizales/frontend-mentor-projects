- CSS Grid

### What I learned

With this project I learned to:
- Implement @font-face instead of @import for the custom font.
- Fluid typography using clamp()
- More responsive images by using <picture> and <source> to adjust the image to the view.
- Advanced flexbox
- Creative UI styling with pseudo-elements

I want to learn other tools to build website, as for right now I'm interested on Tailwind CSS and I'm also learning React.
